Module: pages/Landing

Página de bienvenida (Landing Page) para usuarios no registrados. Presenta la propuesta de valor de CookFlow con una introducción visual, problemas comunes relacionados con la cocina y una sección de recetas destacadas. Esta página actúa como punto de entrada para que los usuarios nuevos entiendan el objetivo de la plataforma y se registren para empezar a usarla. Componentes utilizados: - Button: Botón reutilizable - Card: Vista individual de receta - useFavorites: Hook para gestionar recetas favoritas - mockRecipes: Datos simulados de recetas destacadas Navegación: - Los botones redirigen a la ruta `/signup` para incentivar el registro.
